Domain

As part of an onboarding or induction process, you should be presented with username and password.  The machine you have been allocated should already be on the network.

Most companies- when setting up  new user - set a special flag, that the user must change their password when they next log in.  Be prepared for the possibility that you will need to change your password.

If windows does not prompt you to change your password it is a good idea to change it anyway.

 

Changing your password

Once logged in,

  • press Ctrl + Alt + Del to bring up a blue menu screen
  • Select Change Password
  • Enter your current password into the Old password box
  • Enter the new password into the New Password box
  • Enter the new password into the Confirm Password 
  • Click "go"....  stuff happens and your password is changed.
