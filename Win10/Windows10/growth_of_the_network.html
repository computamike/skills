So when a company first starts out - as a sole trader - then it might be that a company has a single computer, and as there is only 1 person in the company there is no need for a network

Workgroup

As the workforce increases, then the company might need to buy additional computers - but providing each computer with a printer (for example) can prove expensive, so in this scenario, a resource can be shared within the workgroup.

Each person on the network is responsible for sharing their files / folders with fellow members of the workgroup.
